# herdr

herdr is a AI agent multiplexer CLI tool.

## Installation

``` numberSource
brew install herdr
```

## Quick Start

Start the Herdr server with:

``` numberSource
herdr
```

To detach the client do:

`ctrl+b q`

To actually stop the server:

``` numberSource
herdr server stop
```
